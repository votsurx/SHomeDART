# 🏠 SHome v3.0 — Умный дом на Flutter

Локальный контроллер умного дома для устройств Tuya с видео-подсистемой, облачной синхронизацией и MQTT-интеграцией.

---

## 🆕 Что нового в v3.0

### 🔐 Полное шифрование бекапов
- Весь JSON бекапа шифруется (XOR + Base64)
- Пароль облака и токен VK зашифрованы внутри
- Автосохранение MQTT, облака, VK при экспорте
- При импорте — автоматическое расшифровка и восстановление
- Имя файла: `backup_ГГММДД_ЧЧММ.json`

### 🦾 Универсальное составное устройство
- **Один тип — все сложные устройства**: пылесос, кондиционер, увлажнитель
- **Редактор dps_map** — добавление/изменение/удаление DPS через GUI
- **Выбор иконок** для каждого DPS
- **Динамический поллер** — чтение всех DPS из карты
- **Динамический виджет** — генерируется из dps_map
- Сортировка DPS по номерам

### 🔍 Объединённый сканер
- **4 вкладки**: 📡 Tuya | 📹 Камеры (RTSP+ONVIF) | 🔌 Sonoff
- Сканер камер перенесён из Видео в общий сканер

### 🐛 Исправления
- Удаление комнат с переносом устройств в UI
- Безопасное восстановление из бекапа (все поля проверяются на null)
- Красный экран в списке событий
- Имя файла бекапа теперь с датой и временем

## 📱 Основные возможности

### 🏠 Умный дом
- Управление устройствами Tuya по локальной сети
- Поддержка розеток, выключателей (1/2/3 канала), датчиков, штор, ламп
- Комнаты с фильтрацией
- Сцены и таймеры
- Энергомониторинг

### 🎨 Интерфейс
- Устройства на главном экране
- Адаптивная сетка (2-5 колонок)
- Drag & Drop с сохранением порядка
- Погода и время в AppBar (Open-Meteo API)
- Тёмная тема

### 📹 Видеонаблюдение
- Камера планшета/телефона в сетке (фронт/тыл)
- RTSP камеры с превью
- Сканер RTSP/ONVIF камер
- Тревоги Frigate через MQTT
- VK уведомления о тревогах

### ☁️ Облако Mail.ru
- WebDAV подключение
- Автосинхронизация бекапов при сворачивании
- Ротация: 5 последних копий в облаке, 3 локально

### 💾 Резервное копирование
- Локальный автобекап
- Экспорт через Share Sheet (почта, Telegram, облако)
- Импорт через File Picker
- Восстановление из локального бекапа

### 🔔 Уведомления
- VK Bot — пуш при тревогах
- MQTT интеграция с Frigate NVR
- Тестовая кнопка для проверки уведомлений

📦 Установка
Скачайте app-release.apk из Releases

Требования:

Android 7.0+

Планшет/телефон с 4+ ГБ ОЗУ

Для видео: камера или RTSP камеры в сети

🚀 Быстрый старт
Установите APK

Пройдите онбординг (комнаты)

Добавьте устройства Tuya (Device ID, Local Key, IP)

Для видео: Видео → RTSP → Добавить камеру

Для облака: Облако → логин + пароль приложения Mail.ru

Для уведомлений: Настройки → VK → токен + ID

🤖 Составное устройство
Добавьте устройство → Тип: «🦾 Универсальное устройство»

Шестерёнка → DPS карта → Добавить DPS

Настройте DPS: номер, название, тип, роль, иконку

Виджет автоматически построится из dps_map

Подходит для: пылесосов, кондиционеров, увлажнителей, очистителей воздуха.
